Detailed Description

A Piece-wise Affine Warp The ideas for this piece-wise affine triangular warping are taken from the Active appearance models revisited by Iain Matthews and Simon Baker in IJCV 2004 This is used for both validation of landmark detection, and for avatar animation

The code is based on the CLM tracker by Jason Saragih et al.
