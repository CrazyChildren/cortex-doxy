◆ slotOpenForceSensor

void VisualServoAlgorithm::slotOpenForceSensor ( )
slot

slot of RobotManagement::signalOpenForceSensor, which was emited when open forceSensor button clicked.

first open forcesensor, init kd_force, kp_force.. then if ifForceOpen is true, read forceSensor data to forcevalue, devide force by 10, devide torque by 1000. and print devided value. last, put these value to a eigen vector force_initialForceV. then print this vector.

Warning
this function determine the ifUseForce flag, though it's unnecessary. and you can overload force_measure() method, this function are not elegent.
