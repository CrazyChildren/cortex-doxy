detectLargestObject 识别图片中的一个物体，例如最大的脸，将结果储存到‘argestObject’中， 可以用Haar cascades或LBP cascades来做面部检测，或者眼睛、嘴巴、 骑车等。

Parameters
img输入图片
cascade分类器，如面部、眼睛、嘴巴等
largestObject物体检测的返回参数
scalewidth该参数越小，检测速度越快，240就足够找到面部
